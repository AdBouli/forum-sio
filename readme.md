Forum SIO
=========

Pré-requis:
-----------
* Apache2
* PHP 5.5
* MySQL-Server 5.6

Installation
------------
Installez un serveur http Apache2 avec les modules suivants: rewrite, session.

Installez le preprocesseur PHP5.

Installer un serveur de base de données mySQL 5.

Base de données:
----------------
Les paramètres de connexion à la base de données sont modifiables dans 'conf/database.conf'.

Executez le fichier 'datas/sql/structure.sql' dans votre SGBD pour pouvoir utiliser votre application.

Vous pouvez également executez le fichier 'datas/sql/donnees_tests.sql' pour avoir un jeu de test.

Connectez vous en tant que root (identifiants ci-dessous) gràce au lien 'Connexion' dans le coin supérieur gauche de la page d'accueil.

Ensuite, vous avez accès à l'onglet 'Administration' qui s'affichera en haut de l'application dans la barre de navigation du forum.

Administrateur:
* Identifiant: root
* Mot de passe: mdproot

A votre première connexion, changez de mot de passe en allant dans 'Mon compte' puis 'Changer mot de passe'.
